# Push-Up Form Checker

A minimalist web fitness app using computer vision to validate push-up form with instant, meme-style feedback.

## Features

- Real-time push-up form validation using MediaPipe pose detection
- Instant audio feedback for bad form
- Motivational audio every 10 correct reps
- Session statistics tracking
- Zero onboarding friction

## Tech Stack

- Next.js 14
- TypeScript
- React (Function Components)
- Tailwind CSS
- MediaPipe Tasks Vision
- MongoDB Atlas

## Getting Started

```bash
npm install
npm run dev
```

## Testing

```bash
npm test          # Unit tests
npm run test:integration  # Integration tests
npm run test:e2e  # E2E tests
```
